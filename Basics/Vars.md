###### Variables
- System variables
	- created and maintained by OS
	- written in capitals
	- $BASH
- User defined variables
	- usually lowercase
	- var name start with letter or _ 
- must assign vars w no spaces btwn equal sign
	- num1 = 2 does not work
	- num1=2 works
```bash
name=Mark
echo the name is $name

# arrays
array=(1,2,3)
```
